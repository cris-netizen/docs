Aquí encuentras todos los atajos de teclado disponibles en Puente OS. Esta página cubre la navegación global y cada componente individual — consúltala siempre que quieras moverte más rápido por el workspace.

<Note>
  En Mac usa `Cmd`. En Windows y Linux usa `Ctrl`. Cuando veas `Cmd/Ctrl`, aplica el que corresponda a tu sistema operativo.
</Note>

## Navegación global

| Atajo | Acción |
|---|---|
| `Cmd/Ctrl + K` | Abre el buscador global |
| `Cmd/Ctrl + /` | Abre el menú de atajos disponibles |
| `G` luego `H` | Ir al Home del menú |
| `G` luego `P` | Ir a Projects |
| `G` luego `T` | Ir a Tareas |
| `Esc` | Cerrar modal o panel activo |

## Menú — Proyectos y tareas

| Atajo | Acción |
|---|---|
| `C` | Crear nueva tarea |
| `Cmd/Ctrl + Enter` | Guardar tarea y cerrar |
| `Cmd/Ctrl + Shift + N` | Crear nuevo proyecto |
| `J` / `K` | Navegar tarjetas en Kanban (siguiente / anterior) |
| `1` ... `5` | Cambiar a estado: Backlog, Todo, In Progress, Done, Cancelled |

## App Builder

| Atajo | Acción |
|---|---|
| `Enter` (en chat) | Enviar prompt |
| `Shift + Enter` (en chat) | Salto de línea |
| `Cmd/Ctrl + S` | Guardar y recompilar |
| `Cmd/Ctrl + B` | Build manual |
| `Cmd/Ctrl + E` | Activar/desactivar modo Inspect |
| `Cmd/Ctrl + Shift + C` | Copiar código del archivo activo |

## Tables

| Atajo | Acción |
|---|---|
| `Enter` | Editar celda seleccionada |
| `Tab` | Mover al campo siguiente al editar |
| `Shift + Tab` | Mover al campo anterior |
| `Esc` | Cancelar edición de celda |
| `Cmd/Ctrl + F` | Abrir búsqueda global |
| `Cmd/Ctrl + Shift + F` | Abrir filtros |
| `Cmd/Ctrl + N` | Agregar nueva fila |
| `Cmd/Ctrl + Shift + E` | Exportar a CSV |

## Workflows

| Atajo | Acción |
|---|---|
| `Cmd/Ctrl + S` | Guardar el flujo |
| `Cmd/Ctrl + Enter` | Ejecutar el flujo completo |
| `Delete` / `Backspace` | Eliminar nodo o conexión seleccionada |
| `Cmd/Ctrl + Z` | Deshacer |
| `Cmd/Ctrl + Shift + Z` | Rehacer |
| `Cmd/Ctrl + D` | Duplicar nodo |
| `Cmd/Ctrl + /` | Abrir asistente IA (chat) |
| `Space` (presiona y arrastra) | Mover el canvas |

## Agents

| Atajo | Acción |
|---|---|
| `Enter` (en chat de prueba) | Enviar mensaje |
| `Shift + Enter` (en chat) | Salto de línea |
| `Cmd/Ctrl + S` | Guardar agente |
| `Cmd/Ctrl + W` | Abrir configuración del widget |
| `Cmd/Ctrl + L` | Limpiar chat de prueba |

## Tips de productividad
